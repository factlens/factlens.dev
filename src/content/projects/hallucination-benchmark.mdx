An open evaluation dataset for hallucination detection methods.

212 question-response pairs across 7 knowledge domains (finance, medical, science, history, technology, law, geography). Each pair has a grounded response and a human-crafted confabulation — invented institutions, fabricated mechanisms, fictional terminology written by humans, not generated by an LLM.

This distinction matters. Standard benchmarks use LLM-generated fabrications, which carry stylistic artifacts that detection methods learn to exploit. Human-crafted confabulations test whether a method detects genuine semantic drift, not just stylistic anomalies.

Key findings from the accompanying paper:

- DGI achieves 0.958 AUROC on LLM-generated confabulations
- Human-crafted confabulations reveal the true performance boundary
- Cross-domain detection collapses to chance (mean cosine similarity of discriminative directions: -0.07)
- A single global direction achieves 0.96 AUROC on human-crafted data with only 3.8% degradation across domains
